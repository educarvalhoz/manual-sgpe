    Alunos por Projeto

     

    A tela Alunos por Projeto permite ao usuário consultar rapidamente quais estudantes estão vinculados a um projeto específico. Esta funcionalidade é crucial para o gerenciamento de bolsas, a conferência de participantes e a obtenção de documentos anexados durante o processo de inscrição do aluno no projeto.

    Campos de Pesquisa

    O primeiro passo é selecionar o projeto a ser consultado:

    Selecione um Projeto: Campo de seleção que exibe a lista de projetos ativos no sistema.

     

    Pesquisar Alunos: Botão que aciona a busca e carrega a lista de estudantes vinculados ao projeto selecionado.

    Informações na Lista

     

    Após a pesquisa, a tabela Alunos Inscritos no Projeto é exibida com as seguintes colunas:

    Aluno: Nome completo do estudante inscrito no projeto.

    Projeto: Título do projeto ao qual o aluno está vinculado.

    Histórico Escolar: Ícone de Download para obter o histórico escolar anexado pelo aluno.

    Currículo: Ícone de Download para obter o currículo (CV) anexado pelo aluno.

    Documento de Identificação: Ícone de Download para obter a cópia do documento de identificação anexada.

    Tipo: Indica o tipo de participação do aluno no projeto (ex.: Bolsista, Voluntário).

    Passo a Passo

    1. Acesse o menu lateral e clique em Alunos por projeto.

    2. No campo Selecione um Projeto, clique e escolha o título do projeto desejado (ex.: "Monitoria em Informática").

    3. Clique no botão Pesquisar Alunos.

    4. A lista Alunos Inscritos no Projeto será carregada.

    5. Para baixar documentos específicos do aluno (Histórico, Currículo ou Documento de Identificação), clique no respectivo ícone de Download na linha do estudante.
